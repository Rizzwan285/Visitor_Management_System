## Shadcn UI Configuration
When initializing Shadcn UI on a fresh clone, use these settings:
- **Style**: Default
- **Base Color**: Slate
- **CSS Variables**: Yes
- **Component Output Path**: `components/ui`

Run: `npx shadcn@latest init`

## Package Manager
This project uses **npm**.

## Environment Variables
Copy `.env.local.example` to `.env.local` and fill in all values before running the app.
